Programming assignment 2

Replace this HTML markup with markup for your game. Your game does not need to literally be a rock scissors paper game but it must do the following:

  1. Prompt the user for an input that is then saved as a variable
  2. Have at least 1 element that is saved as a constant
  3. Compare 2 or more strings using a comparison operator
  4. Use an 'if / else' conditional statement
  5. Have images that appear based on those conditions
  6. Feature an AI player that makes a choice by generating a random number
  7. Your program should be user friendly: if the player inputs a string that the program does not recognize, it should ask the player to try again and case should not matter. (i.e.: your program should recognize 'Rock', 'ROCK', and 'rock' as the same word.)
  8. Place your JavaScript code in a separate .js file

Your code should also not display any errors in the console.
